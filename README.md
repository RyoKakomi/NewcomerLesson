# NewcomerLesson
新人用の遊び場

好きなものをコピペでもクローンでも好きに使ってください。

基本的に各LessonのREADME.mdに出題内容を記載する予定なので、それを参考に問題を解いてみてください。
全てjavaScriptで出題しますが、ほかの言語に置き換えても面白いかもしれません。

**question.jsのコードを修正して、テストコードの出力結果がすべてtrueになるようにしてください。**

https://paiza.io/ja
